## Screenshot Guidelines

- Use PNG format for best quality
- Capture at a reasonable resolution (1920x1080 or similar)
- Include relevant UI elements and context
- Blur or redact any sensitive information
- Use consistent browser/theme across all screenshots
- Ensure text is readable at various sizes
